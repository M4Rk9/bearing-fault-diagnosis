# Notebooks

Use this folder for step-by-step experimentation.

Suggested notebook order:

1. `01_data_loading.ipynb` - inspect CWRU `.mat` files.
2. `02_signal_visualization.ipynb` - plot waveform, FFT, and spectrogram.
3. `03_feature_extraction.ipynb` - test RMS, kurtosis, skewness, crest factor, FFT and envelope features.
4. `04_ml_models.ipynb` - train SVM and Random Forest.
5. `05_cnn_model.ipynb` - train 1D-CNN on raw signal windows.

Keep heavy outputs and dataset files out of git.
